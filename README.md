# Apple Product Showcase Website

A stunning, modern Apple-inspired product showcase website built with React, Three.js, and GSAP animations. This project features smooth scrolling animations, 3D product viewers, and a sleek UI design that mirrors Apple's premium aesthetic.

## âœ¨ Features

- **3D Product Viewer**: Interactive 3D models powered by Three.js and React Three Fiber
- **Smooth Animations**: Scroll-triggered animations using GSAP and ScrollTrigger
- **Responsive Design**: Fully responsive layout built with Tailwind CSS
- **Modern UI**: Apple-inspired minimalist design
- **Performance Optimized**: Built with Vite for lightning-fast development and builds
- **State Management**: Efficient state handling with Zustand
